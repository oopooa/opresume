# 更新日志 (Changelog)



## v0.1.3 (2026-03-15)

### ✨ 新特性 (Features)

* 新增 Bug 反馈与新功能请求的 GitHub Issue 模板
* 新增 Issue 模板选择器配置


## v0.1.2 (2026-03-15)

### ✨ 新特性 (Features)

* 新增应用版本号展示：于顶部 Logo 旁直观显示当前应用版本。
* 优化编辑区体验：将顶部工具栏设为固定悬浮（Sticky），在滚动长简历时也可随时操作菜单。

### 🐛 Bug 修复 (Bug Fixes)

* 修复生产环境头像上传失败问题：改用 Base64 纯前端直存方案，剥离对外部图床或服务器的依赖。
* 优化本地存储性能：配合 Base64 方案，将头像上传尺寸限制下调至 500KB，有效防止导出 JSON 文件体积异常膨胀。

## v0.1.1 (2026-03-14)

### ✨ 新特性 (Features)

* 新增数据清空功能：方便用户一键清空当前简历数据，从零开始编辑。

### 🐛 Bug 修复 (Bug Fixes)

* 修复生产环境下保存到 localStorage 实际未生效问题。
